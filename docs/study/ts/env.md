# typeScript 开发环境

- [typescriptlang](https://www.typescriptlang.org/zh/play?#)

## 全局安装 ts-node typescript

- [ts-node](https://www.npmjs.com/package/ts-node)

```bash
npm i ts-node typescript -g
```

## 初始化 ts 项目

```bash
tsc --init
```

## 将 ts 文件编译成 js 文件

```bash
tsc xxx.ts
```

## ts-node 运行 ts 文件

```bash
ts-node xxx.ts
```
